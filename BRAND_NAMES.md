# 專案品牌與 UI 設計指引

本專案正式選定品牌名稱為：**Binnacle**

## 1. 品牌核心概念
* **名稱由來**：大航海時代船隻甲板上用來保護指南針與航海日誌（Logbook）的防潮、防震、防外力干擾的「羅經箱」。
* **產品寓意**：Binnacle 是 AI 時代的合規安全箱。無論外部風浪（監管變更、安全威脅）多大，協議的核心治理記憶與審計證據（Walrus Blobs）都在此受到絕對保護，不可篡改且隨時可供驗證。

---

## 2. UI 設計指引 (Style Guide)

為展現機構級的信賴感與 Sui 生態特色，視覺設計應契合「精密航海儀器」與「深海安全」意象：

### A. 色彩計畫 (Color Palette)
* **深海主色 (Deep Ocean Blue)**: `#0A1128` 至 `#001F54`
  * *用途*：背景與主要區塊。象徵 Walrus 巨大、安全且深邃的去中心化數據存儲。
* **精密銅金色 (Antique Brass/Bronze)**: `#D4AF37` / `#C5A059`
  * *用途*：強調色、邊框、圖示。代表實體羅經箱的黃銅材質，隱喻時間耐用度、歷史厚重感與精準度。
* **信任綠色 (Trust Emerald/Teal)**: `#00F2FE` / `#10B981`
  * *用途*：驗證成功狀態、安全通道、動態掃描信號。代表密碼學驗證（Seal & Sui Move）的無暇狀態。

### B. 字型設計 (Typography)
* **標題**：採用 `Space Grotesk` 或 `Outfit` (展現儀器儀表板的現代感與科技感)。
* **內文與日誌**：採用 `Inter` 搭配 `JetBrains Mono` (強調日誌數據的清晰易讀與技術嚴謹度)。

### C. 視覺元素與微互動 (Visuals & Micro-interactions)
* **玻璃擬態 (Glassmorphism)**: 使用高透明度的毛玻璃效果作為資訊卡片背景，模擬「水下觀測窗」與「保護罩」的感覺。
* **驗證動畫 (Verification Sweep)**: 當審計 Agent 進行資料檢索或驗證 Walrus blob hash 時，UI 呈現黃銅指針偏轉、或者如雷達般的圓形偏光掃描動畫，象徵儀器正在對深海數據進行測量（Fathom）。
* **狀態圖示**：採用精密刻度、船舵鎖扣、密封艙門等機械細節，取代常見的扁平化 Web3 圖標。
